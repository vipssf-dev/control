User Interface

Main Processes:

  • 1.0 Add Exam: Enter exam data (Day, Date, Grade, Subject, Source)
  • 2.0 Display Exams: Retrieve and display all exams sorted by day and grade
  • 3.0 Update Progress: Update tracking step status (Photocopying, Grading, Review...)
  • 4.0 Delete Exam: Remove an exam record from the system

Data Store:

  • PostgreSQL Database: Stores all exam data and progress status

Data Flows:

  • Input: Exam details (Day, Hijri Date, Grade Level, Subject, Question Source)
  • Processing: CRUD operations on exam records
  • Output: Sorted exam list with progress indicators
